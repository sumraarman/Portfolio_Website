# Portfolio Website with Flask

## Overview

This project is a simple personal portfolio website developed using Python Flask, HTML, and CSS. It displays personal information, skills, and a contact form.

## Features

- Home page
- About Me section
- Skills section
- Contact form
- Simple and responsive design

## Technologies Used

- Python
- Flask
- HTML
- CSS

## Project Structure

```
Portfolio_Website/
│
├── app.py
├── README.md
│
├── templates/
│   └── index.html
│
└── static/
    └── style.css
```

## How to Run

1. Install Flask

```
pip install flask
```

2. Run the application

```
python app.py
```

3. Open your browser and visit

```
http://127.0.0.1:5000/
```

## Project Outcome

This project demonstrates the basics of Flask web development, HTML templates, CSS styling, and handling a simple contact form.
